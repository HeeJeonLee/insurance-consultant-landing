# 이희전 보험상담사 랜딩페이지

25년 경력 금융보험 전문가의 펫보험 전문 상담 서비스

## 🎯 주요 기능

- ✅ 전문가 프로필 소개
- ✅ 5개 보험사 펫보험 상품 안내
- ✅ AI 챗봇 24/7 자동 상담
- ✅ 비대면 상담 신청
- ✅ 모바일 최적화
- ✅ Google Analytics 연동

- Vite
- Tailwind CSS

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
